- **Phase B** Indicator engines (analytics/<domain>) — incremental
- **Phase C** Shared scenario engine — core delivered, extended in v0.4.0
- **Phase D** Enterprise services (auth, reporting, GIS, API, monitoring)
- **Phase E** Disease modules: UHC, Maternal (integrated), HIV (delivered), TB, Malaria, NCD, RHIS, Digital Health, SRHR, SDG 3
